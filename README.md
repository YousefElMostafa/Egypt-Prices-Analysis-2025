## 📌 Project Overview
This project analyzes food and service prices in Egypt using official CAPMAS data.

---

## 🎯 Objectives
- Extract data from PDF to Excel
- Clean and transform data
- Analyze prices using Excel
- Build interactive Power BI Dashboard
- Generate insights and recommendations

---

## 🛠️ Tools Used
- Excel
- Power BI
- GitHub

---

## 📚 Dataset Source
CAPMAS - Central Agency for Public Mobilization and Statistics (Egypt)
